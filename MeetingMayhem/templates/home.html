Mission:

You are a malicious adversary trying to disrupt the agents' plans on when and where they will meet to exchange an essential asset. To do so, you may insert, block, or modify messages sent over the network, which you control. You, however, cannot defeat encryption and signatures that are properly formed using keys you do not know. At the end of the game session, the two agents will vote to decide a time and place to meet. You will try to prevent the agents from reaching consensus on the meeting time and place.

Goal:

Do not allow the agents to agree on the time and place to meet.

Cryptographic Tools:

  • Delete Any message.
  • Modify any Plaintext (non-encrypted) message that is not signed.
  • Send a message to any recipient pretending to be someone else.
  • Symmetric Encryption: Symmetrically Encrypt or Symmetrically decrypt any message using keys you know.
  • Asymmetric Encryption: Asymmetrically Encrypt any message using the Receiver’s Public Key, or asymmetrically decrypt any message using any Private Key you know.
  • Signature: Sign any message with any Sender’s Private Key you know; the receiver can verify the identity the sender's identity using the Sender’s Public Key. Here, a valid signature also encrypts and guarantees message integrity (i.e., that the message has not been modified in transit).
